| UR-002 | 로그인 | 등록된 계정 존재 | ❍ 로그인 페이지 접근<br>❍ 이메일, 비밀번호 입력<br>❍ 로그인 버튼 클릭<br>❍ bcrypt 비밀번호 검증<br>❍ JWT 토큰 발급<br>❍ 역할별 페이지 이동 (USER→/home, ADMIN→/dashboard) |
| UR-003 | 로그아웃 | 로그인 상태 | ❍ 사이드바 로그아웃 버튼 클릭<br>❍ localStorage 토큰 및 사용자 정보 삭제<br>❍ 메인 페이지(/) 또는 로그인 페이지로 이동 |
| UR-004 | 개인정보 수정 | 로그인 상태 | ❍ 마이페이지 접근<br>❍ 수정할 정보 변경 (이름, 연락처, 성별, 부서)<br>❍ 저장 버튼 클릭<br>❍ 변경사항 DB 반영<br>❍ 사이드바 사용자 정보 즉시 갱신 |
| UR-005 | 개인정보 조회 | 로그인 상태 | ❍ 홈 또는 마이페이지 접근<br>❍ 사용자 정보 표시 (이름, 이메일, 부서, 연락처, 성별)<br>❍ 홈에서 최근 활동 및 통계 표시 |

## 2. AI 챗봇

| 요구사항고유번호 | 요구사항명칭 | 사전조건 | 세부내용 |
|-----------------|-------------|----------|----------|
| CR-001 | 대화 세션 관리 | USER 권한으로 로그인 | ❍ 챗봇 페이지 접근<br>❍ 새 대화 세션 생성<br>❍ 세션 목록 조회 (최신순 정렬)<br>❍ 세션 선택 시 대화 이력 로드<br>❍ 세션 제목 수정<br>❍ 세션 삭제 (소프트 삭제 - ARCHIVED) |
| CR-002 | AI 대화 | 대화 세션 선택 또는 생성 | ❍ 메시지 입력 후 전송<br>❍ Redis에서 세션 컨텍스트 조회 (요약 + 최근 10개 메시지)<br>❍ ChromaDB에서 관련 문서 검색 (RAG)<br>❍ AI 엔진에 프롬프트 + 컨텍스트 전달<br>❍ 커스텀 스트리밍으로 실시간 응답 표시 (TEXT_DATA:/DOCS_DATA: 형식)<br>❍ 대화 내역 DB 저장<br>❍ Redis 컨텍스트 캐시 업데이트 (TTL 1시간) |
| CR-003 | 참고 문서 표시 | RAG 검색 결과 존재 | ❍ AI 응답과 함께 참고 문서 정보 수신<br>❍ 문서명, 관련도 표시<br>❍ 문서 클릭 시 상세 정보 또는 원본 접근 |
